# PowerPlan

## Description
Projet DotnetCore 6

## Prérequis
```
- Docker
- Docker Compose
- DotnetCore
```

## Getting Started <a name = "getting_started"></a>

### Installing

at the root of the project: Write next command


```
docker-compose up -d
```
the project PowerPlan crée a container and run this one when
the container runded you can check URL
